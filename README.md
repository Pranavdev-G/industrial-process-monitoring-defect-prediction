# Industrial Process Monitoring Defect Prediction

## Overview
This project focuses on industrial process monitoring and defect prediction using machine learning techniques.

## Features
- Data preprocessing
- Defect prediction
- Industrial monitoring
- Machine learning model training
- Visualization and analysis

## Tech Stack
- Python
- FastAPI
- Machine Learning
- Pandas
- NumPy
- Scikit-learn

## Installation

```bash
pip install -r requirements.txt
```

## Run Project

```bash
python -m uvicorn main:app --reload
```
